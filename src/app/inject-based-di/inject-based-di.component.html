Inject-Based Dependency Injection

Angular has introduced a new way of dependency injection. This new way is uses the inject function. The inject function "offers more accurate types and better compatibility with standard decorators, compared to constructor-based injection." (official quote)
Old way 
          @Component()
          export class WidgetComponent {{ '{' }}
            constructor(private _service: SomeService){{ '{}' }}
          {{ '}' }}
New way 
          @Component()
          export class WidgetComponent {{ '{' }}
            private _service = inject(SomeService);
          {{ '}' }}
