Image Optimization

This one's a big topic to get into, so this will just be an overview. Angular provides the NgOptimizedImage directive to help with image loading. It does this in a few ways:

  • Allowing you to specify which images have priority (Largest Contentful Paint images)
  • Lazily loading other images by default
  • Automatically generating tags and attriubtes to aid in performant image loading
  • Requiring size attributes (helps prevent layout shift)
  • Warning you if an image will be distorted
